# Adz Dzikra — Website Resmi

> Buku Pelajaran Menulis Huruf Al-Qur'an karya M. Siffin AlMufti, S.Ag

Website resmi Adz Dzikra — satu-satunya toko yang langsung dari tangan penulis.

## 🛠️ Tech Stack

- **HTML** + **CSS** + **Vanilla JS** (no framework)
- **Google Fonts**: Plus Jakarta Sans, Noto Naskh Arabic
- **Icons**: Lucide Icons (CDN)
- **Hosting**: Cloudflare Pages

## 📁 Project Structure

```
├── index.html          # Single-page website
├── styles.css          # Main stylesheet
├── script.js           # Main JavaScript
├── assets/             # Images, icons, favicon
│   ├── images/         # Product photos, author photo
│   └── favicon/        # Favicon files
└── README.md
```

## 🚀 Development

Cukup buka `index.html` di browser, atau gunakan live server:

```bash
# Dengan Python
python3 -m http.server 8000

# Dengan Node.js
npx serve .
```
